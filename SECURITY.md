# Security Policy

## Reporting a vulnerability

Email **security@zaragoza.se** or **info@zaragoza.se** with the subject line starting with "[SECURITY]".

We will respond within **7 business days**. For documentation / data repos, "security" also includes:
- Factual errors with legal/financial impact
- Outdated regulatory references that could mislead readers
- Template flaws that could create contractual exposure

## Supported versions

The latest `main` branch is actively maintained. Older tagged releases receive corrections for critical issues only.
